<Tabs>

**Since**: `v1.0.0`
:::note
- This rule is recommended by Biome. A diagnostic error will appear when linting your code.
- This rule has an **unsafe** fix.
:::

Avoid using unnecessary `continue`.

## Examples

### Invalid

```js
loop: for (let i = 0; i < 5; i++) {
  continue loop;
}
```

### Valid

```js
while (i) {
  if (i > 5) {
    continue;
  }
  console.log(i);
  i--;
}

loop: while (1) {
  forLoop: for (let i = 0; i < 5; i++) {
    if (someCondition) {
      continue loop;
    }
  }
}
```
