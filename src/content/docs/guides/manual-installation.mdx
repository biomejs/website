Learn how to install Biome manually on all major platforms.

## macOS

Biome is available for macOS (Intel and Apple Silicon) as a standalone binary.

### Installation script

Run the following command in your terminal to install the latest stable version.

```shell frame=none lines=false showLineNumbers=false
curl -fsSL https://biomejs.dev/install | sh
```

### Homebrew

Biome is available as a [Homebrew formula](https://formulae.brew.sh/formula/biome) for macOS users.

```shell frame=none lines=false showLineNumbers=false
brew install biome
```

## Linux

Biome is available for Linux (x86 and ARM) as a standalone binary.

### Installation script

Run the following command in your terminal to install the latest stable version.

```shell frame=none lines=false showLineNumbers=false
curl -fsSL https://biomejs.dev/install | sh
```

## Windows

Biome is available for Windows (x86 and ARM) as a standalone binary.

:::note
Use the Linux variant for Windows Subsystem for Linux (WSL).
:::

### Installation script

Run the following command in your PowerShell to install the latest stable version.

```powershell frame=none lines=false showLineNumbers=false
powershell -c "irm biomejs.dev/install.ps1 | iex"
```

## Docker

Biome is available as a [Docker image] starting from version `v1.7.0`.

[Docker image]: https://github.com/biomejs/docker/pkgs/container/biome

```
ghcr.io/biomejs/biome:{major}
ghcr.io/biomejs/biome:{major}.{minor}
ghcr.io/biomejs/biome:{major}.{minor}.{patch}
```

Here are a couple examples on how to use the Docker image:

:::note
The default workdir is `/code` in the Docker image.
:::

```shell
# Lint files
docker run -v $(pwd):/code ghcr.io/biomejs/biome lint
docker run -v $(pwd):/code ghcr.io/biomejs/biome lint --write

# Format files
docker run -v $(pwd):/code ghcr.io/biomejs/biome format
docker run -v $(pwd):/code ghcr.io/biomejs/biome format --write
```
