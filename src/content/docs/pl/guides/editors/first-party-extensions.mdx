Są to rozszerzenia, które są utrzymywane przez zespół Biome i są częścią [organizacji Biome](https://github.com/biomejs).

### VS Code

Integracja edytora Biome pozwala na:

* Formatowanie plików przy zapisie lub podczas wydawania polecenia _Format_.
* Lintowanie plików i stosowanie poprawek kodu

Zainstaluj nasze oficjalne [rozszerzenie Biome dla VS Code](https://marketplace.visualstudio.com/items?itemName=biomejs.biome) z Visual Studio Marketplace.

Aby uczynić Biome domyślnym formaterem, otwórz [obsługiwany plik](/internals/language-support/) i:

1. otwórz *paletę poleceń* (View lub <kbd>Ctrl</kbd>/<kbd title="Cmd">⌘</kbd>+<kbd title="Shift">⇧</kbd>+<kbd>P</kbd>)
1. wybierz *Format Document With...*
1. wybierz *Configure Default Formatter*
1. wybierz *Biome*.

Przeczytaj więcej na [stronie referencyjnej](/reference/vscode).

### IntelliJ

Aby zainstalować wtyczkę Biome IntelliJ, przejdź na [oficjalną stronę wtyczki](https://plugins.jetbrains.com/plugin/22761-biome) lub wykonaj następujące kroki:

**Z IDE JetBrains:**

1. Otwórz IntelliJ IDEA.
2. Przejdź do **Settings/Preferences**.
3. Wybierz **Plugins** z menu po lewej stronie.
4. Kliknij na zakładkę **Marketplace**.
5. Wyszukaj "Biome" i kliknij **Install**.
6. Uruchom ponownie IDE, aby aktywować wtyczkę.

**Z dysku:**

1. Pobierz plik .zip rozszerzenia z zakładki Versions.
2. Naciśnij <kbd title="Apple">⌘</kbd> / <kbd>Ctrl</kbd>+<kbd>Alt</kbd>+<kbd>S</kbd> + <kbd title="Comma">,</kbd>, aby otworzyć ustawienia IDE, a następnie wybierz Plugins.
3. Na stronie rozszerzenia kliknij przycisk The Settings, a następnie kliknij Install Plugin from Disk….

### Zed

1. otwórz *paletę poleceń* (View lub <kbd>Ctrl</kbd>/<kbd title="Cmd">⌘</kbd>+<kbd title="Shift">⇧</kbd>+<kbd>P</kbd>)
2. wybierz **zed: extensions**
3. wyszukaj **Biome**
4. wybierz **Install**

Przeczytaj więcej na [stronie referencyjnej](/reference/zed).
