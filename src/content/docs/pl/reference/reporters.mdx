CLI Biome akceptuje argument `--reporter`, który pozwala zmienić sposób drukowania diagnostyk i podsumowania do terminala.

## JSON

:::caution
Ten reporter jest eksperymentalny i może ulec zmianom w wydaniach patch.
:::

Emituje podsumowanie i diagnostyki w formacie JSON.

```shell
biome ci --reporter=json
```

## JSON pretty

:::caution
Ten reporter jest eksperymentalny i może ulec zmianom w wydaniach patch.
:::

Tak samo jak `--reporter=json`, emituje podsumowanie i diagnostyki w formacie JSON, a wyjście jest formatowane przy użyciu bieżących opcji formatowania JSON (plik konfiguracyjny lub domyślne).

```shell
biome ci --reporter=json-pretty
```

## GitHub

Użyj tego reportera w workflow GitHub. Gdy właściwie skonfigurowany w workflow PR, GitHub wydrukuje wiadomość dla każdej wyemitowanej informacji/ostrzeżenia/błędu.

```shell
biome ci --reporter=github
```

```
::error title=lint/suspicious/noDoubleEquals,file=main.ts,line=4,endLine=4,col=3,endColumn=5::Use === instead of ==
::error title=lint/suspicious/noDebugger,file=main.ts,line=6,endLine=6,col=1,endColumn=9::This is an unexpected use of the debugger statement.
::error title=lint/nursery/noEvolvingAny,file=main.ts,line=8,endLine=8,col=5,endColumn=6::This variable's type is not allowed to evolve implicitly, leading to potential any types.
```
