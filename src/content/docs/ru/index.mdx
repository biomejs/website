<div class="hero-code">
    <h2 class="head">Исправляйте проблемы, осваивайте лучшие практики</h2>

    **Biome — это [производительный линтер](https://github.com/biomejs/biome/tree/main/benchmark#linting)** для _JavaScript_,
    _TypeScript_, _JSX_, _CSS_ и _GraphQL_, который предоставляет **[<NumberOfRules/> правил](/ru/linter/)** из ESLint,
    TypeScript ESLint и [других источников](/ru/linter/rules-sources/).

    **Biome выдаёт подробную диагностику, учитывающую контекст**, которая помогает вам улучшить ваш код
    и стать более хорошим программистом!

    <LinterExample/>

    Попробуйте линтер Biome в [песочнице](/playground/) или напрямую в вашем проекте:

    ```shell
    npm i -D --save-exact @biomejs/biome
    npx @biomejs/biome lint --write ./src
    ```

</div>


<div class="hero-code">
    <h2 class="head">Всё в одном</h2>

    Вы можете не только отдельно форматировать и анализировать ваш код, но и делать **всё это с помощью одной команды**!

    Каждый инструмент бесшовно интегрируется с другими для создания **целостного инструментария** для веб-проектов.

    Запустите все инструменты с помощью команды `check`:

    ```shell
    npm i -D --save-exact @biomejs/biome
    npx @biomejs/biome check --write ./src
    ```

</div>
