Это расширения, которые поддерживаются командой Biome и являются частью [организации Biome](https://github.com/biomejs).

### VS Code

Интеграция редактора Biome позволяет вам:

* форматировать файлы при сохранении или при вызове команды _Format_.
* проверять файлы линтером и применять исправления кода

Установите наше официальное [расширение Biome для VS Code](https://marketplace.visualstudio.com/items?itemName=biomejs.biome) из Visual Studio Marketplace.

Чтобы сделать Biome форматером по умолчанию, откройте [поддерживаемый файл](/internals/language-support/) и:

1. откройте *Command Palette* (View или <kbd>Ctrl</kbd>/<kbd title="Cmd">⌘</kbd>+<kbd title="Shift">⇧</kbd>+<kbd>P</kbd>)
1. выберите *Format Document With...*
1. выберите *Configure Default Formatter*
1. выберите *Biome*.

Подробнее читайте на [справочной странице](/reference/vscode).

### IntelliJ

Чтобы установить плагин Biome для IntelliJ, перейдите на [официальную страницу плагина](https://plugins.jetbrains.com/plugin/22761-biome) или следуйте этим шагам:

**Из JetBrains IDEs:**

1. Откройте IntelliJ IDEA.
2. Перейдите в **Settings/Preferences**.
3. Выберите **Plugins** из левого меню.
4. Нажмите на вкладку **Marketplace**.
5. Найдите "Biome" и нажмите **Install**.
6. Перезапустите IDE для активации плагина.

**С диска:**

1. Скачайте расширение .zip с вкладки Versions.
2. Нажмите <kbd title="Apple">⌘</kbd> / <kbd>Ctrl</kbd>+<kbd>Alt</kbd>+<kbd>S</kbd> + <kbd title="Comma">,</kbd> чтобы открыть настройки IDE, затем выберите Plugins.
3. На странице расширения нажмите кнопку Settings, затем нажмите Install Plugin from Disk….

### Zed

1. откройте *Command Palette* (View или <kbd>Ctrl</kbd>/<kbd title="Cmd">⌘</kbd>+<kbd title="Shift">⇧</kbd>+<kbd>P</kbd>)
2. выберите **zed: extensions**
3. найдите **Biome**
4. выберите **Install**

Подробнее читайте на [справочной странице](/reference/zed).
