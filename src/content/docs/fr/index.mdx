<div class="hero-code">
    <h2 class="head">Résolvez les problèmes, apprenez les bonnes pratiques</h2>

    **Biome est un [outil de linting performant](https://github.com/biomejs/biome/tree/main/benchmark#linting)** pour _JavaScript_, _TypeScript_, _JSX_, _CSS_ et _GraphQL_ qui comprend **[<NumberOfRules/> règles](/fr/linter/rules/)** d’ESLint, de TypeScript ESLint et d’[autres sources](/fr/linter/rules-sources/).

    **Biome fournit des diagnostics détaillés et contextualisés** qui vous aident à améliorer votre code et à devenir un meilleur programmeur&nbsp;!

    <LinterExample />

    Essayez l’outil de linting de Biome dans le [bac à sable](/playground/) ou directement dans votre projet&nbsp;:

    ```shell
    npm i -D --save-exact @biomejs/biome
    npx @biomejs/biome lint --write ./src
    ```

</div>


<div class="hero-code">
    <h2 class="head">Tout en même temps</h2>

    Non seulement vous pouvez formater et linter votre code séparément, mais vous pouvez le faire **en même temps avec une seule commande&nbsp;!**

    Les outils s’intègrent les uns aux autres pour créer **un ensemble d’outils cohérent** pour les projets web.

    Exécutez tous les outils avec la commande `check`&nbsp;:

    ```shell
    npm i -D --save-exact @biomejs/biome
    npx @biomejs/biome check --write ./src
    ```

</div>
