Se trata de extensiones mantenidas por el equipo de Biome y que forman parte de la [organización Biome](https://github.com/biomejs).

### VS Code

La integración del editor Biome te permite:

- Formatear archivos al guardar o al emitir el comando _Format_.
- Verificar errores y aplicar correcciones de código

Instala nuestra extensión oficial [Biome VS Code extension](https://marketplace.visualstudio.com/items?itemName=biomejs.biome) desde Visual Studio Marketplace.

Para que Biome sea el formateador por defecto abre un [archivo soportado](/internals/language-support/) y:

1. abre la _Paleta de comandos_ (o <kbd>Ctrl</kbd>/<kbd title="Cmd">⌘</kbd>+<kbd title="Shift">⇧</kbd>+<kbd>P</kbd>)
1. selecciona _Format Document With..._
1. selecciona _Configure Default Formatter_
1. selecciona _Biome_.

Más información en la [página de referencia](/reference/vscode).

### IntelliJ

Para instalar el plugin Biome IntelliJ, ve a la [página oficial del plugin](https://plugins.jetbrains.com/plugin/22761-biome) o sigue estos pasos:

**De JetBrains IDEs:**

1. Abre IntelliJ IDEA.
2. Ve a **Settings/Preferences**.
3. Selecciona **Plugins** en el menú de la izquierda.
4. Haz clic en la pestaña **Marketplace**.
5. Busca "Biome" y haz clic en **Install**.
6. Reinicia el IDE para activar el plugin.

**De disco:**

1. Descarga la extensión .zip desde la pestaña Versiones.
2. Pulsa `⌘Сmd,` para abrir la configuración del IDE y luego selecciona Plugins.
3. En la página de extensiones, haz clic en el botón Configuración y, a continuación, en Instalar complemento desde disco....

### Zed

1. abre la _Paleta de comandos_ (o <kbd>Ctrl</kbd>/<kbd title="Cmd">⌘</kbd>+<kbd title="Shift">⇧</kbd>+<kbd>P</kbd>)
2. selecciona **zed: extensions**
3. selecciona **Biome**
4. selecciona **Install**

Más información en la [página de referencia](/reference/zed).
