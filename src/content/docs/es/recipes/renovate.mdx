[Renovate](https://docs.renovatebot.com/) es una herramienta popular para la actualización automática de dependencias.

## biome.json

Renovate tiene una [regla preestablecida compartida](https://docs.renovatebot.com/presets-customManagers/#custommanagersbiomeversions) que puede ayudar a mantener la versión de `$schema` sincronizada y actualizada dentro de los archivos de configuración `biome.json`.

Para utilizarlo, añade `customManagers:biomeVersions` a tu lista de `extends`.

```json title="renovate.json"
{
  "$schema": "https://docs.renovatebot.com/renovate-schema.json",
  "extends": ["customManagers:biomeVersions"]
}
```
