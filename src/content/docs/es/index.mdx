<div class="hero-code">
    <h2 class="head">Corrije problemas y aprende las mejores prácticas</h2>

    **Biome es un [linter de alto rendimiento](https://github.com/biomejs/biome/tree/main/benchmark#linting)** para _JavaScript_,
    _TypeScript_, _JSX_, _CSS_ y _GraphQL_ que cuenta con **[<NumberOfRules/> reglas](/linter/rules/)** de ESLint,
    TypeScript ESLint, y [otras fuentes](/linter/rules-sources/).

    **Biome produce diagnósticos detallados y contextualizados** que te ayudaran a mejorar tu código y a convertirte en un mejor programador!

    <LinterExample/>

    Prueba el Biome linter en el [playground](/playground/) o directamente en tu proyecto:

    ```shell
    npm i -D --save-exact @biomejs/biome
    npx @biomejs/biome lint --write ./src
    ```

</div>

<div class="hero-code">
    <h2 class="head">Todo a la vez</h2>

    No sólo puedes formatear y analizar tu codigo por separado, sino que puedes hacerlo **todo a la vez con un solo comando**!

    Cada herramienta se integra perfectamente con las demás para crear una **cadena de herramientas cohesionada** para proyectos web.

    Ejecuta todas las herramientas con el comando `check`:

    ```shell
    npm i -D --save-exact @biomejs/biome
    npx @biomejs/biome check --write ./src
    ```

</div>
