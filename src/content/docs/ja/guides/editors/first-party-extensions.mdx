これらはBiomeチームによって管理されており、[Biome オーガニゼーション](https://github.com/biomejs) の一部です。

### VS Code

Biomeエディタ統合を使用すると、次のことができます：

- ファイルの保存時または*format*コマンドの発行時にファイルをフォーマットする。
- ファイルをリントし、コード修正を適用する。

公式の [Biome VS Code 拡張機能](https://marketplace.visualstudio.com/items?itemName=biomejs.biome)をVisual Studio Marketplaceからインストールしてください。

Biomeをデフォルトのフォーマッタにするには、[サポートされているファイル](/internals/language-support/)を開き、次の手順に従います：

1. _コマンドパレット_ を開きます（ビューまたは<kbd>Ctrl</kbd>/<kbd title="Cmd">⌘</kbd>+<kbd title="Shift">⇧</kbd>+<kbd>P</kbd>）。
2. _ドキュメントのフォーマット方法を選択..._ を選びます。
3. _デフォルトフォーマッタを設定_ を選びます。
4. _Biome_ を選択します。

詳細は[リファレンスページ](/reference/vscode)をご覧ください。

### IntelliJ

Biome IntelliJプラグインをインストールするには、[公式プラグインページ](https://plugins.jetbrains.com/plugin/22761-biome)にアクセスするか、以下の手順に従ってください：

**JetBrains IDE から：**

1. IntelliJ IDEAを開きます。
2. **設定/環境設定**に移動します。
3. 左側のメニューから**プラグイン**を選択します。
4. **マーケットプレイス**タブをクリックします。
5. "Biome" を検索して**インストール**をクリックします。
6. プラグインを有効にするためにIDEを再起動します。

**ディスクから：**

1. バージョンタブから拡張機能の .zipをダウンロードします。
2. `⌘Cmd,`を押してIDE設定を開き、プラグインを選択します。
3. 拡張機能ページで設定ボタンをクリックし、"ディスクからプラグインをインストール ..."を選択します。

### Zed

1. *コマンドパレット*を開きます（ビューまたは<kbd>Ctrl</kbd>/<kbd title="Cmd">⌘</kbd>+<kbd title="Shift">⇧</kbd>+<kbd>P</kbd>）。
2. **zed: extensions** を選択します。
3. **Biome** を検索します。
4. **インストール**を選択します。

詳細は[リファレンスページ](/reference/zed)をご覧ください。
