[Renovate](https://docs.renovatebot.com/) 是一种流行的自动依赖项更新工具。

## biome.json

Renovate 有一个 [共享预设规则](https://docs.renovatebot.com/presets-customManagers/#custommanagersbiomeversions)，这可以帮助保持 `$schema` 在 `biome.json` 配置文件的版本同步且保持为最新。

为了使用它，添加 `customManagers:biomeVersions` 到您的 `extends` 列表。


```json title="renovate.json"
{
  "$schema": "https://docs.renovatebot.com/renovate-schema.json",
  "extends": ["customManagers:biomeVersions"]
}
```
